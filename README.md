# SemistandardTableaux.jl

A Julia package for working with semistandard Young tableaux (SSYT) and
Schur polynomials.

## Installation

```julia
using Pkg
Pkg.add(url="https://github.com/pseudoeffective/SemistandardTableaux.jl")
```

## Usage

```julia
using SemistandardTableaux
```
